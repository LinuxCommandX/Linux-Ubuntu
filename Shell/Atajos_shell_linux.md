# Listado de Atajos esenciales para la Shell de Linux

A continuación, se presenta un listado de algunos atajos interesantes y altamente eficientes para la shell de Linux. Te recomiendo que los pruebes y selecciones los que te resulten más cómodos para optimizar tu flujo de trabajo en el día a día.

---

## 📌 Atajos de Teclado (Shell Shortcuts)

| Atajo | Acción / Descripción |
| :--- | :--- |
| **`Ctrl + A`** | Ir al **principio** de la línea en la que se está escribiendo. |
| **`Ctrl + E`** | Ir al **final** de la línea en la que se está escribiendo. |
| **`Ctrl + L`** | **Borra la pantalla**, con un comportamiento similar al comando `clear`. |
| **`Ctrl + U`** | Borra la línea **anterior** a la posición del cursor. Si está al final, borra toda la línea. |
| **`Ctrl + H`** | Borra el **carácter anterior** a la posición del cursor (igual que la tecla Retroceso). |
| **`Ctrl + R`** | Permite **buscar** un comando en el historial de uso (búsqueda inversa). |
| **`Ctrl + C`** | **Interrumpe o cancela** la ejecución del programa o comando actual. |
| **`Ctrl + D`** | **Cierra** la shell o terminal actual (envía un EOF). |
| **`Ctrl + W`** | **Corta la palabra anterior** a la posición del cursor. |
| **`Ctrl + K`** | **Corta la línea siguiente** a la posición del cursor. |
| **`Ctrl + T`** | Intercambia los dos últimos **caracteres** anteriores a la posición del cursor. |
| **`Esc + T`** | Intercambia las dos últimas **palabras** anteriores a la posición del cursor. |
| **`Alt + .`** o **`!$`** | Hace referencia al **último argumento** del comando ejecutado anteriormente. |
| **`Alt + F`** | **Avanza** el cursor una palabra hacia adelante en la línea actual. |
| **`Alt + B`** | **Retrocede** el cursor una palabra hacia atrás en la línea actual. |
| **`Alt + l` / `u`** | Convierte la palabra actual a **minúsculas (`l`)** o **mayúsculas (`u`)**. |
| **`Tab`** | **Autocompleta** nombres de comandos, archivos y carpetas automáticamente. |

---

> 💡 **Tip de productividad:** Combinar el uso de `Ctrl + A` / `Ctrl + E` con `Alt + F` / `Alt + B` te permite moverte por comandos largos en la terminal en segundos sin necesidad de usar las flechas del teclado.
